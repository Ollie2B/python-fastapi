# python-fastapi
Template for REST API made with Python using FastApi framework and MongoDB.

# Dependencies

  - MongoDB running at //localhost:27017
  - Fastapi
  - Pymongo
  - Uvicorn

# Running

`uvicorn server:app --reload`

 - Base URL: `http://localhost:8000/`
 - Swagger URL: `http://localhost:8000/docs`
